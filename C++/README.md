### コンパイル方法

```bash
$ cd 対象のディレクトリ
$ g++ {対象ファイル名} && ./a.out
```

### 標準入力がある場合

- 上記コマンド入力後に待機状態になるのでそのまま入力

```bash
  # ex) 下記の場合、
  # int N, v;
  # cin >> N >> v;
  # vector<int> a(N);

  # for (int i = 0; i < N; ++i) cin >> a[i];
  # for (int i = 0; i < N; ++i) {
  #  cout << a[i] << endl;
  # }

  $ g++ {対象ファイル名} && ./a.out
  # このまま待機状態になるので下記を入力
  5 3
  1
  2
  3
  4
  5
```
